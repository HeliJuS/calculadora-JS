## Introdução
Caculadora simples capaz de realizar as quatro operações básicas da matemática. 

## Funcionalidades

* Somar
* Subtrair
* Multiplicar
* Dividir
* Copiar o resultado para área de trnaferência
* Alternar tema da página entre modo escuro e modo claro
